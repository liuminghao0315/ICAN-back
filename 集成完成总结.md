## 如何使用

### 步骤1：配置七牛云密钥
1. 复制示例配置文件：
   ```bash
   copy .env.example .env
   ```

2. 编辑 `.env` 文件，填入七牛云的AccessKey和SecretKey：
   ```
   QINIU_ACCESS_KEY=您的AccessKey
   QINIU_SECRET_KEY=您的SecretKey
   ```

### 步骤2：启动算法模拟器
```bash
# Windows
start_qiniu_algorithm.bat

# Linux/Mac
python qiniu_algorithm_simulator.py
```

### 步骤3：发送测试任务
通过Java端发送视频分析任务，系统将自动调用七牛云API进行处理。
