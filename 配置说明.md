# 后端配置说明

后端使用两种配置文件：
1. **Java 配置：** `application.properties` (Spring Boot 标准配置)
2. **Python 配置：** `config.py` (Python 脚本配置)

---

## 一、Java 配置 (application.properties)

### 配置文件位置
`backend/src/main/resources/application.properties`

### 需要修改的配置项

#### 1. 数据库配置
```properties
spring.datasource.url=jdbc:mysql://你的数据库地址:3306/ican?serverTimezone=GMT%2B8
spring.datasource.username=你的数据库用户名
spring.datasource.password=你的数据库密码
```

#### 2. Redis 配置
```properties
spring.data.redis.host=你的Redis地址
spring.data.redis.port=6379
spring.data.redis.password=你的Redis密码
```

#### 3. 邮件配置（用于发送验证码）

**QQ 邮箱：**
```properties
qqMail.sender=你的QQ邮箱
qqMail.senderName=SynSight Security Center
qqMail.smtpCode=你的QQ邮箱SMTP授权码
```

**网易邮箱：**
```properties
netease.sender=你的163邮箱
netease.senderName=SynSight Security Center
netease.smtpCode=你的163邮箱SMTP授权码
```

#### 4. MinIO 对象存储配置
```properties
minio.endpoint=http://你的MinIO地址:9000
minio.accessKey=minioadmin
minio.secretKey=你的MinIO密码
minio.bucketName=ican-videos
```

#### 5. RabbitMQ 配置
```properties
spring.rabbitmq.host=你的RabbitMQ地址
spring.rabbitmq.port=5672
spring.rabbitmq.username=guest
spring.rabbitmq.password=guest
```

#### 6. yt-dlp 配置（视频下载）
```properties
# Windows 环境
download.ytdlp-path=D:/yt-dlp/yt-dlp.exe

# Linux 环境
download.ytdlp-path=yt-dlp
```

#### 7. FFmpeg 配置（缩略图生成）
```properties
# Windows 环境
thumbnail.ffmpeg-path=C:/ProgramData/chocolatey/bin/ffmpeg.exe

# Linux 环境
thumbnail.ffmpeg-path=ffmpeg
```

## 二、Python 配置 (config.py)

### 配置文件位置
`backend/config.py`

### 需要修改的配置项

#### DeepSeek API 配置

用于视频分析时生成详细证据（身份判定、涉及高校、内容主题、对学校态度、潜在舆论风险、处置建议）。

```python
class Config:
    # DeepSeek API Key - 必填
    # 获取方式：访问 https://platform.deepseek.com/ 注册并获取 API Key
    DEEPSEEK_API_KEY = "填写你的 DeepSeek API Key"

    # 以下配置一般不需要修改
    DEEPSEEK_API_URL = "https://api.deepseek.com/v1/chat/completions"
    DEEPSEEK_MODEL = "deepseek-chat"
    DEEPSEEK_TIMEOUT = 45
    DEEPSEEK_MIN_TRANSCRIPTION_LENGTH = 20
```

**如何获取 DeepSeek API Key：**
1. 访问 https://platform.deepseek.com/
2. 注册/登录账号
3. 进入 API Keys 页面
4. 创建新的 API Key
5. 复制 API Key 并填入 `config.py`

## 其他配置（一般不需要修改）

- `server.port=8080` - 后端服务端口
- `jwt-access-token-expire-minutes=30` - Access Token 过期时间（分钟）
- `jwt-refresh-token-expire-days=7` - Refresh Token 过期时间（天）
- `upload.chunk-size=5242880` - 分片上传大小（5MB）
- `spring.servlet.multipart.max-file-size=500MB` - 最大文件上传大小
